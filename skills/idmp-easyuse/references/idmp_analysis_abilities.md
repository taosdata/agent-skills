# IDMP 分析能力参考

IDMP 分析挂载在元素上，表单分四段：基本信息 → 触发器 → 计算 → 事件。

---

## 一、基本信息

| 字段 | 说明 |
|---|---|
| 名称（必填） | 分析唯一名称 |
| 类别 | 可选分类标签 |
| 创建后立即启用 | 默认勾选，保存后立即运行 |
| 对乱序数据重新计算 | 接收迟到数据时重算受影响窗口 |
| 描述 | 自由文本说明 |

---

## 二、触发类型

除**定时窗口**外，其余触发类型要求属性数据引用类型为 TDengine Metric。

### 1. 滑动窗口
按事件时间以固定间隔滚动触发。

| 参数 | 说明 |
|---|---|
| 滑动时长 | 两次触发间的时间间隔（如 1min、10s） |

适用：实时滚动聚合（移动均值、滚动累计、滑动 KPI）。

### 2. 异常检测
按滑动时间计划对目标属性运行 AI 异常检测算法，无需设定阈值。

| 参数 | 说明 |
|---|---|
| 滑动时长 | 运行频率 |
| 目标（必填） | 待分析属性，多选则自动创建子分析 |
| 算法（必填） | 异常检测算法 |
| 白噪声数据检查 | 数据为白噪声时跳过检查 |
| 算法参数 | 可选算法特定参数，格式 `a=1,b=2` |

适用：无法预定义异常形态、固定阈值误报多、需早期预警。

### 3. 定时窗口
基于系统时钟按固定间隔触发，与数据到达无关。

| 参数 | 说明 |
|---|---|
| 定时 | 时间间隔（如每 1h、每 1d） |
| 偏移 | 边界后延迟触发（如 6h 偏移 = 每天 06:00 触发） |

适用：固定时间报表/汇总（按小时/天/班次）。

### 4. 数据输入
每当新数据写入指定属性（或任意属性）时触发。

| 参数 | 说明 |
|---|---|
| 目标 | 触发属性，留空则任意属性有新数据即触发 |

适用：逐条实时计算、单位换算、派生属性、即时限值检查。

### 5. 状态窗口
整型属性值从一个状态切换到另一个状态时触发。

| 参数 | 说明 |
|---|---|
| 状态（必填） | 监控状态变化的整型属性，多选则自动创建子分析 |

适用：按运行模式统计、OEE 可用率、批次汇总（批次号自增）。

### 6. 事件窗口
基于用户定义的开始/结束条件表达式触发，条件针对元素属性计算。

| 参数 | 说明 |
|---|---|
| 开始触发 — 表达式（必填） | 结果为正值时开启窗口 |
| 开始触发 — 持续时长 | 条件需持续满足的最短时长，防止噪声误触发 |
| 结束触发 — 表达式（必填） | 结果为正值时关闭窗口并触发计算 |

适用：检测持续性状况（超限、压缩机喘振、低 OEE 区间），捕获窗口内完整特征。

### 7. 会话窗口
元素在指定不活动时间内无新数据到达时触发，计算覆盖此前活跃会话。

| 参数 | 说明 |
|---|---|
| 不活动间隔 | 数据静默多久后认为会话结束 |

适用：数据突发式上报（车辆行程、批次过程、CNC 加工任务）。

### 8. 计数窗口
新写入记录数达到指定数量时触发。

| 参数 | 说明 |
|---|---|
| 目标 | 计数的具体属性，留空则统计所有属性 |
| 计数 | 触发所需累积记录数 |

适用：基于样本数量的统计过程控制（SPC）、实验室仪器批次分析。

### 公共字段（所有触发类型共享）

| 字段 | 说明 |
|---|---|
| 预过滤 | 触发评估前的数据过滤表达式，仅满足条件的行参与计算 |
| 重算历史 | 启用后对历史数据回填输出属性；可配置优先重算历史和起始时间 |

## 三、计算

### 1. 计算对象

| 选项 | 说明 |
|---|---|
| 元素本身 | 针对当前元素自身属性计算 |
| 子元素聚合 | 对共享同一模板的子元素跨元素聚合（需指定子元素模板，可选子表过滤） |

### 2. 窗口聚合

- **启用（默认）**：在时间窗口上聚合，需配置**间隔**（如 1h）。
- **禁用**：逐行/单点计算，适用于行级转换（单位换算、派生计算）。

### 3. 输出时间戳

| 选项 | 说明 |
|---|---|
| 窗口开始 | 以窗口起始时间作为输出时间戳 |
| 窗口结束（默认） | 以窗口结束时间作为输出时间戳 |

可配置偏移量（数字 + 单位，默认 0s）对齐显示。

### 4. 输出属性

每行包含：

| 列 | 说明 |
|---|---|
| 表达式 | 计算表达式（通过表达式编辑器填写） |
| 元素属性 | 结果写入的元素属性（时序历史） |
| 事件属性 | （仅启用事件生成时可见）事件触发时捕获计算值的事件属性 |

支持多行，单次分析可计算多个指标并写入不同属性。
